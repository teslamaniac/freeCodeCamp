## Select Tag

The '<select>' tag creates a drop down list. Items can be added with the `<option>` tag.
  
#### Example
```html

<select>
  <option>Ice Cream</option>
  <option>Chocolate</option>
  <option>Potatoes</option>
  <option>Lollies</option>
</select>

````
